# agentic-systems

> Five complete, standalone agent systems you can run, not five more snippets.

## Problem

The gap between "I understand the patterns" and "I shipped a system" is where most learning
stalls. You need end-to-end examples, not fragments.

## Architecture

5 complete, self-contained agent systems, each composing the [patterns](agentic-patterns.md)
into something that does a real job from input to output. The step between learning patterns
and shipping a [SaaS](agentic-saas-boilerplate.md).

## Lessons learned

- Composition is the actual skill; individual patterns are table stakes.
- Standalone-runnable beats a shared framework for *learning* - no hidden machinery.

## Demo

Each system runs on its own; pick one and run it end to end.
