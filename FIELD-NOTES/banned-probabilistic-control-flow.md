# Why I Banned Probabilistic Control Flow in My Agent Systems

> Letting an LLM decide what to do next wins on Twitter and fails in production: infinite
> loops, latency spikes, exploded token budgets. Control flow belongs in deterministic code.
> The model proposes; a DAG state machine with Kahn's topological sort disposes.
